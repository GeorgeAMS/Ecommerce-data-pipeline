# 🚀 Ecommerce Data Pipeline (ETL)

Este proyecto implementa un pipeline de datos robusto que extrae información de productos desde una API externa, la transforma para su análisis y la almacena en una base de datos relacional.

## 📋 Descripción
El objetivo de este repositorio es demostrar un flujo **ETL (Extract, Transform, Load)** completo. Automatizamos la obtención de datos de un catálogo de comercio electrónico para generar insights sobre precios, categorías y valoración de productos.

## 🏗️ Arquitectura del Proyecto
*   **Fuente de Datos:** [FakeStoreAPI](https://fakestoreapi.com) (REST API).
*   **Extracción y Transformación:** Script en **Python** utilizando `requests` y `pandas`.
*   **Almacenamiento:** Base de datos relacional **PostgreSQL**.
*   **Visualización:** Dashboard interactivo en **Power BI**.

## 🛠️ Stack Tecnológico
*   **Lenguaje:** Python 
*   **Librerías:** `requests`, `pandas`, `sqlalchemy`, `psycopg2`.
*   **Base de Datos:** PostgreSQL.
*   **BI:** Power BI Desktop.
